四个月编程刻意练习

Web开发从0到全栈

你可以学到

  • 第一阶段（1~5周）

    网页开发基础

    • 了解前端开发基本概况，并能熟练使用常用的开发工具
    • 掌握HTML中常用的标签以及每个标签的语义
    • 理解前端页面布局常用方法，并能够用CSS+HTML实现常用页面布局
    • 熟练使用JavaScript语言，并能够使用JavaScript完成页面常见特效

  • 第二阶段（6~10周）

    Node.js

    • 能够理解客户端/服务器、动态网站以及网络通信等前后端开发相关概念
    • 掌握 Node.js 服务端 JavaScript 的基础知识
    • 掌握数据库的基本概念，及数据库的常用操作
    • 掌握 MongoDB 的常用操作，包含建数据库，表的增删改查等操作

  • 第三阶段（11~15周）

    JavaScript 全栈框架实战

    • 能够用 React 实现前端组件，进行页面布局
    • 掌握用 React 实现前后端交互的方法
    • 理解 Restful 接口思想
    • 能够使用 Express 构建 API
    • 能够使用 Express 和 MongoDB 建立通信，通过接口进行数据库相关操作

  • 毕业设计（4周内）

    项目实战+大比赛

你不能错过的4个理由

助教陪伴成长

我们的助教将提供给你实时的辅导，及时解决你在学习上遇到的问题，核查代码，精准答疑，不用担心自己的疑惑得不到回复。 助教也会同你一起规划时间，制定任务，监督学习，不用担心自己是一个人从而难以坚持下去，你的程序员成长轨迹一定有他的陪伴。

最前沿的软件开发技术

HTML5, ES2015, Node.js, React.js, Redux.js, Git, REST API，MongoDB 我们以行业最前沿的技术为工具，教你最有效的编程思维

结识热爱编程的小伙伴

在这里，你不再是一个人学习，而是会有一群和你一样热爱编程的小伙伴，大家一起学习、一起讨论问题，还会定期线下聚会碰面，一起享受编程的快乐。

实战项目经验

最后阶段你和你所属的团队会协作完成一次软件项目的开发，100%还原真实项目挑战。我们还会举办编程比赛，挑战自己的编程极限。

常见问题

第五期特训营有什么特色呢？
在形式上，第五期特训营的课程将会分为三个阶段，每个阶段为五周时间，在每个阶段完成后均会休息一周时间，使大家进行总结、回顾和状态的调整；
在内容上，第五期特性营更加注重对全栈技能和工程实践的培养，加入了更多的关于Node服务端和数据库相关的知识，且在每个阶段结束时均有一个阶段大作业，以强化学员的动手实践能力。
在时间上，第五期特训营紧的助教们针对前四期的遇到的问题总结经验，给学员提供更优质的答疑，辅导等服务。
助教是如何分配的呢？
在分配助教的时候首先会基于地理位置来分配，方便大家进行线下聚会；特训营共有5大战区，分别是西安、武汉、成都、北京、空军，若不在前四个城市，会分配到空军战区，以独特的方式进行聚会学习。
